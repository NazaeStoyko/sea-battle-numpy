# Sea Battle (NumPy)

## Опис

Консольна реалізація гри «Морський бій».
Гравець грає проти бота. Поля генеруються автоматично, бот стріляє випадковими координатами без повторів.
Результати пострілів відображаються у вигляді текстового поля.

---

## Функціонал

* Генерація ігрового поля
* Випадкове розміщення кораблів
* Валідація розміщення (без перетинів)
* Чергування ходів
* Повторний хід бота при влучанні
* Приховане поле бота
* Відображення гри в консолі

---

## Структура проєкту

```
.
├── field.py          # логіка поля і кораблів
├── navigation.py     # допоміжна логіка розміщення
├── shot_bot.py       # генерація пострілів бота
├── main.py           # основний запуск гри
└── .gitignore
```

## Керування

Під час гри потрібно вводити координати:

```
row: <номер рядка>
col: <номер стовпця>
```

---

## Позначення клітинок

```
0 — порожня клітинка
1 — корабель
2 — зона навколо корабля
3 — промах
4 — влучання
```

---

## Логіка гри

* Гравець і бот ходять по черзі
* Якщо постріл влучний — гравець/бот ходить ще раз
* Якщо промах — хід переходить іншому
* Гра закінчується, коли всі кораблі знищені
